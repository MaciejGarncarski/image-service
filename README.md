# Image service

Upload images to your server! And serve them to your users!

## Features

- File upload
- Static image serving
- API key authentication

## Installation

### Fill out the .env file

```dotenv
KEY=
IMAGE_DIR=
APP_DIR=
PORT=
API_KEY=
```

### Install dependencies

```bash
pnpm install
```

## Usage

```bash
pnpm dev
```
